# Clustering and PageRank

## Project Description

This repository contains implementations of core Big Data and Information Retrieval concepts developed as part of an academic assignment. The project is divided into three practical modules that demonstrate how data can be grouped, searched, and ranked using real-world techniques.

### Modules Included

## Part 1: Clustering

Implemented clustering techniques on the UCI Spam dataset to group similar records based on feature patterns. This demonstrates unsupervised learning concepts used in customer segmentation, anomaly detection, and pattern discovery.

### Key Tasks:

* Data preprocessing
* Similarity measurement
* Cluster formation
* Result interpretation

---

## Part 2: Web Search Engine

Built a basic search engine using an inverted index and positional indexing. The system retrieves webpages containing a keyword and also returns the positions of words inside documents.

### Key Features:

* Keyword search
* Multi-document retrieval
* Positional indexing
* Fast lookup using indexing
* Text preprocessing and token normalization

---

## Part 3: PageRank

Implemented the PageRank algorithm to rank webpages based on link structure and importance. This simulates how search engines prioritize authoritative pages.

### Key Features:

* Graph construction
* Iterative rank updates
* Handling dangling nodes
* Sorting pages by score
* Authority-based ranking
